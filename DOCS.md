# Go2Study Bot Documentation

## Recent Changes

### Database Schema Update (2024-05-20)
- Recreated the tasks table to ensure proper support for image-based questions
- Added support for storing image paths and quantitative characteristics
- Fixed issue with missing image_path column in tasks table

## Новые типы вопросов и поддержка изображений

### Добавлена поддержка двух типов вопросов:
1. Стандартные вопросы (standard) - текстовые вопросы с вариантами ответов
2. Вопросы с количественными характеристиками (quantitative) - вопросы, сравнивающие две характеристики (А и Б) с вариантами ответов A, B, C, D

### Поддержка изображений:
- Добавлена возможность прикреплять изображения к вопросам
- Изображения извлекаются из PDF-файлов и сохраняются локально
- При отображении вопроса с изображением, оно отправляется как фото с подписью, содержащей текст вопроса

### Изменения в базе данных:
- Добавлены новые поля в таблицу tasks:
  - question_type: тип вопроса (standard/quantitative)
  - image_path: путь к изображению вопроса
  - characteristic_a: характеристика А для количественных вопросов
  - characteristic_b: характеристика Б для количественных вопросов
  - source_file: исходный файл, из которого был извлечен вопрос

### Обработка PDF-файлов:
- Создан новый модуль pdf_processor.py для извлечения вопросов и изображений из PDF
- Поддерживается извлечение как стандартных, так и количественных вопросов
- Изображения сохраняются в отдельную директорию для каждого PDF-файла

### Зависимости:
- Pillow (PIL) 11.2.1 - для обработки изображений из PDF-файлов
