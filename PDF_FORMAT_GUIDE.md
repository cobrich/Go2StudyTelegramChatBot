# Руководство по формату PDF файлов для Go2Study Bot

## Поддерживаемые форматы

Текущий PDF процессор поддерживает следующие форматы:

### 1. **Базовый формат (рекомендуемый)**
```
ТЕСТ: Математика 5 класс - Базовый уровень

ТЕМА: Операции с дробями и остатками

1. Вычислите: 2,46 × 18
A) 39.05
B) 44.28 ✅
C) 106.48
D) 122.7

2. Найдите результат: 0,253 × 250
A) 63.25 ✅
B) 47.01
C) 83.03
D) 84.06
```

### 2. **Альтернативные форматы (поддерживаются)**

#### Формат с галочкой после ответа:
```
1. 2,46 × 18 =
A) 39.05
B) 88.77 ✅
C) 106.48
D) 122.7
```

#### Формат с галочкой перед ответом:
```
1. Решите уравнение: x + 5 = 12
✅ A) x = 7
B) x = 17
C) x = 2
D) x = 12
```

## Требования к содержанию

### **Обязательные элементы:**
1. **Нумерация вопросов**: 1., 2., 3., ... или 1), 2), 3), ...
2. **Варианты ответов**: A), B), C), D) (поддерживаются и русские А), Б), В), Г))
3. **Маркер правильного ответа**: ✅ (галочка)
4. **Минимум 2 варианта ответа** на вопрос

### **Рекомендуемые элементы:**
1. **Указание темы**: `ТЕМА: [название из списка TOPICS]`
2. **Название теста**: `ТЕСТ: [описание]`
3. **Пустые строки** между вопросами для лучшего парсинга
4. **Четкое форматирование** без лишних символов

## Автоматическое определение тем

Если тема не указана явно, процессор использует AI для определения темы на основе содержания вопроса.

## Языковая поддержка

- **Русский язык**: полная поддержка
- **Казахский язык**: автоматическое определение по символам әіңғүұқөһ

## Что делать при ошибках

Если PDF файл не обрабатывается корректно:

1. **Проверьте формат**: убедитесь, что используется поддерживаемый формат
2. **Проверьте маркеры**: галочки ✅ должны быть четко видны
3. **Проверьте нумерацию**: вопросы должны быть пронумерованы
4. **Проверьте варианты**: должно быть A), B), C), D)
5. **Проверьте темы**: используйте темы из списка выше

## Добавление нового PDF файла

1. Поместите PDF файл в папку `files/`
2. Добавьте путь к файлу в список `pdf_files` в `src/services/pdf_processor.py`
3. Запустите: `python -m src.services.pdf_processor`

## Технические детали

- **Поддерживаемые символы**: латинские и кириллические буквы, цифры, математические символы
- **Максимальная длина ответа**: 60 символов
- **Кодировка**: UTF-8
- **Формат файла**: PDF (любая версия)
