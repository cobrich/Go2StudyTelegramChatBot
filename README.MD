# Go2Study Bot (Telegram-бот для тестирования)

Go2Study Bot — это Telegram-бот, разработанный для помощи пользователям в изучении различных тем (преимущественно математических) путем прохождения интерактивных тестов. Бот генерирует вопросы, проверяет ответы, предоставляет объяснения и отслеживает прогресс пользователя.

## Ключевые возможности

*   **Интерактивное тестирование:**
    *   Выбор тем для изучения.
    *   Получение вопросов с вариантами ответов.
    *   Приоритетная подача вопросов, на которые пользователь ранее отвечал неверно.
    *   Загрузка вопросов из базы данных.
    *   Динамическая генерация новых вопросов с помощью ИИ (Google Gemini), если в базе недостаточно материала.
    *   Мгновенная обратная связь с указанием правильного ответа и объяснения.
    *   Навигация по вопросам теста ("Предыдущий", "Следующий").
*   **Просмотр ранее отвеченных вопросов:** При возврате к уже отвеченному вопросу в рамках текущего теста отображается предыдущий ответ пользователя и объяснение.
*   **Подробные результаты теста:**
    *   Количество правильных ответов и процент выполнения.
    *   Список допущенных ошибок с возможностью запросить детальное объяснение для каждой.
*   **Отслеживание прогресса:**
    *   Общая статистика по пройденным тестам, среднему результату и количеству ошибок.
    *   Список последних пройденных тем с результатами.
*   **Управление состоянием:**
    *   Защита от случайного прерывания теста командами из главного меню.
    *   Очистка состояния при завершении или смене теста.
*   **База данных:**
    *   Хранение тем, вопросов, результатов тестов и ошибок пользователей.
*   **Команда "Помощь"** для получения основной информации о боте.

## Технологии

*   **Python 3.x**
*   **python-telegram-bot:** Библиотека для взаимодействия с Telegram Bot API.
*   **Google Generative AI (Gemini):** Для генерации учебных материалов (вопросов и объяснений).
*   **SQLite:** В качестве базы данных для хранения информации (может быть заменена на другую СУБД).

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL_вашего_репозитория>
    cd <имя_папки_проекта>
    ```

2.  **Создайте и активируйте виртуальное окружение (рекомендуется):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # для Linux/macOS
    # venv\Scripts\activate    # для Windows
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Убедитесь, что у вас есть файл `requirements.txt`, содержащий все необходимые библиотеки, такие как `python-telegram-bot`, `google-generativeai`, `httpx` и др.)*

4.  **Настройте переменные окружения:**
    Боту требуются API-ключи для работы. Обычно их хранят в файле `.env` или устанавливают как переменные окружения системы.
    Создайте файл `.env` в корне проекта со следующим содержимым:

    ```env
    TELEGRAM_BOT_TOKEN="ВАШ_ТЕЛЕГРАМ_БОТ_ТОКЕН"
    GEMINI_API_KEY="ВАШ_GOOGLE_GEMINI_API_КЛЮЧ"
    ```
    Замените `"ВАШ_ТЕЛЕГРАМ_БОТ_ТОКЕН"` и `"ВАШ_GOOGLE_GEMINI_API_КЛЮЧ"` на ваши реальные ключи.

    В коде бота (например, в `bot.py` или отдельном файле конфигурации) необходимо будет загружать эти переменные. Пример с использованием `python-dotenv` (если вы его используете):
    ```python
    # В начале bot.py
    from dotenv import load_dotenv
    import os

    load_dotenv() # Загружает переменные из .env

    TELEGRAM_TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
    GEMINI_API_KEY = os.getenv("GEMINI_API_KEY")
    ```

5.  **Инициализируйте базу данных:**
    Если база данных создается при первом запуске или требует специальной команды для создания таблиц (например, через `db_utils.py`), убедитесь, что это будет выполнено. В текущей структуре, вероятно, `db = Database("quiz_database.db")` и `db.create_tables()` позаботятся об этом.

6.  **Запустите бота:**
    ```bash
    python bot.py
    ```
