# 📚 Go2Study Bot - Telegram Bot для изучения математики

🤖 **Telegram бот для подготовки к экзаменам НИШ (5-6 классы)** с адаптивной системой обучения и ИИ-генерацией вопросов.

## ✨ Ключевые возможности

- 🎯 **Адаптивное тестирование** с фокусом на слабые места
- 📊 **Подробная аналитика** прогресса обучения  
- 🔄 **Повторение ошибок** для закрепления материала
- 👥 **Система администрирования** с whitelist пользователей
- 🌐 **Многоязычность** (русский/казахский)
- 📁 **Импорт вопросов** из PDF с ИИ-объяснениями
- 🎨 **Современный интерфейс** с inline-клавиатурами

## 📚 Структура тем

Бот содержит полную структуру тем для подготовки к НИШ:

### 🔢 Числа и арифметика (7 тем)
- Действия с дробями
- Сравнение чисел  
- Масштаб и проценты
- Соотношения и пропорция
- Арифметические действия и порядок выполнения
- Модуль числа
- Неравенства

### 🔤 Алгебраические выражения (5 тем)
- Упрощение выражений
- Выражения
- Раскрытие скобок
- Линейные уравнения и нахождение неизвестного
- Системы уравнений

### 📐 Геометрия (4 темы)
- Площадь и периметр фигур
- Определение углов
- Масштабы и расстояния
- Круг и окружность

### 📊 Анализ данных и статистика (4 темы)
- Среднее арифметическое
- Числовые последовательности
- Текстовые задачи
- Функции

### 🧠 Логико-математическое мышление (3 темы)
- Задачи на закономерности чисел
- Логические задачи
- Сравнение выражений

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка окружения
Создайте файл `.env` на основе `.env.template`:
```bash
cp .env.template .env
# Отредактируйте .env с вашими ключами
```

### 3. Инициализация базы данных
```bash
# Создание суперадмина
python src/init_superadmin.py

# Создание полной структуры тем
python src/init_topics.py
```

### 4. Запуск бота
```bash
python src/main.py
```

## 🔧 Настройка

### Переменные окружения (.env)
```env
# Telegram Bot
TELEGRAM_BOT_TOKEN=ваш_токен_бота

# Gemini AI для генерации вопросов
GEMINI_API_KEY1=ваш_ключ_gemini

# База данных (Supabase/Neon PostgreSQL)
DATABASE_URL=postgresql://user:pass@host:port/db
USE_POSTGRESQL=true
```

### Создание суперадмина
```bash
python src/init_superadmin.py
```
Введите:
- Telegram user_id
- Username (без @)
- ФИО

### Инициализация тем
```bash
python src/init_topics.py
```
Создаст полную структуру из 46 подтем в 5 основных разделах.

## 👥 Использование

### Для учеников:
1. Запустите бота: `/start`
2. Выберите "📚 Выбрать тему и начать"
3. Выберите основной раздел (например, "🔢 Числа и арифметика")
4. Выберите конкретную тему (например, "Действия с дробями")
5. Проходите тест из 10 вопросов
6. Получайте подробные объяснения к каждому ответу

### Для администраторов:
- `/admin` - доступ к админ-панели
- Управление учениками, темами, вопросами
- Импорт вопросов из PDF файлов
- Просмотр статистики и аналитики

## 🏗️ Архитектура

```
src/
├── main.py              # Точка входа
├── handlers/            # Обработчики команд и callback'ов
│   ├── admin/          # Админ-панель
│   └── ...
├── db/                 # База данных
│   ├── repositories/   # Репозитории для работы с данными
│   └── ...
├── services/           # Бизнес-логика
├── utils/              # Вспомогательные функции
└── config/             # Конфигурация и константы
```

### База данных
- **main_topics** - основные разделы тем
- **subtopics** - конкретные темы для изучения
- **questions** - вопросы с вариантами ответов
- **users** - пользователи с прогрессом
- **admins** - администраторы системы

## 🐳 Развертывание

### Docker
```bash
docker-compose up -d
```

### VPS/Сервер
```bash
# Установка как systemd service
sudo cp go2study-bot.service /etc/systemd/system/
sudo systemctl enable go2study-bot
sudo systemctl start go2study-bot
```

### Cloud платформы
- Heroku: используйте `Procfile`
- Railway/DigitalOcean Apps: автоматическое развертывание

Подробное руководство: [deploy_guide.md](deploy_guide.md)
